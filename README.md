# 🦠 COVID-19 India State-wise Analysis

## 📌 About
This project is an end-to-end exploratory data analysis of India’s COVID-19 state-wise dataset using NumPy and Pandas.  
The analysis focuses on infection trends, recovery rates, mortality rates, and inter-state comparisons without using visualization libraries.

---

## 📊 Dataset
- **Source:** Kaggle — COVID-19 India Dataset (Dhamodharan R)
- **Coverage:** 36 States & Union Territories
- **Features:** 9 columns including cases, deaths, recoveries, etc.

---

## 🎯 Objectives
- Analyze COVID-19 impact across Indian states
- Calculate key health indicators like CFR and Recovery Rate
- Identify high-risk and low-risk states
- Detect statistical patterns and outliers
- Study relationship between cases and deaths

---

## 📈 Key Insights
- Maharashtra recorded the highest total cases (~17% of India’s total)
- India’s overall recovery rate is **98.81%**
- Punjab had the highest Case Fatality Rate (2.69%)
- Mizoram and Arunachal Pradesh showed the lowest CFR values
- Strong correlation (r > 0.9) between total cases and deaths

---

## 🧮 Analysis Performed
- Total, mean, and standard deviation using NumPy
- Case Fatality Rate (CFR) calculation per state
- Recovery rate ranking across states
- Outlier detection using IQR method
- Cases per million population analysis
- Correlation between cases and deaths
